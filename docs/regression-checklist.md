# 全量回归清单（性能/适配/Bug）

## 1. 登录与会话
- [ ] 管理员登录成功，`/api/session` 返回 `authenticated=true`
- [ ] 访客登录可进入演示模式，且无真实写操作
- [ ] 会话过期后自动跳转登录页，无白屏
- [ ] 根管理员令牌（`Authorization` / `X-Admin-Token` / `admin_token`）可生效

## 2. 邮箱生成与切换
- [ ] 随机生成邮箱成功，地址展示正确
- [ ] 随机人名生成邮箱成功
- [ ] 自定义邮箱非法字符拦截正常
- [ ] 历史邮箱点击切换后，当前邮箱高亮同步

## 3. 收件箱（重点）
- [ ] 移动端首次进入收件箱不卡顿（滚动流畅）
- [ ] 自动刷新倒计时正常递减并按周期刷新
- [ ] 手动刷新与自动刷新不会冲突闪烁
- [ ] 列表在数据未变化时不重复重绘（视觉稳定）
- [ ] 翻页（上一页/下一页）无明显掉帧

## 4. 邮件详情（重点）
- [ ] 打开详情弹窗无长时间卡住
- [ ] 验证码点击复制成功（事件委托路径）
- [ ] HTML 邮件 iframe 可渲染，纯文本邮件可显示
- [ ] 关闭弹窗后再次打开无异常状态残留

## 5. 发件箱
- [ ] 发件记录列表加载与翻页正常
- [ ] 发件详情可查看
- [ ] 删除发件记录后列表状态正确刷新

## 6. 管理页与邮箱总览
- [ ] 用户列表加载、搜索、编辑正常
- [ ] 邮箱总览 grid/list 切换正常
- [ ] 批量操作（如有）执行后 UI 状态正确

## 7. 转发与收藏
- [ ] 转发设置弹窗打开、保存、回显正确
- [ ] 收藏/取消收藏状态与图标同步
- [ ] 列表筛选（收藏/转发）结果正确

## 8. 响应式适配
- [ ] 移动端（<=900px）无横向滚动条
- [ ] 移动端历史邮箱/生成邮箱切换稳定
- [ ] PC 端侧栏收起后主区域不位移、不裁切
- [ ] 横屏手机布局可用（按钮无遮挡）

## 9. 部署与变量稳定性
- [ ] 发布后 Dashboard 中 `TEMP_MAIL_DB` / `MAIL_EML` / `MAIL_DOMAIN` / `JWT_TOKEN` 绑定仍存在
- [ ] 未在 `wrangler.toml` 添加 `[vars]`
- [ ] Worker `name` 未被意外改名导致发布到新脚本

## 10. 性能验收建议
- [ ] 移动端收件箱连续滚动 30 秒无明显掉帧
- [ ] 打开邮件详情平均耗时可接受（主观不卡）
- [ ] 自动刷新期间操作（翻页/点详情）无明显卡顿
